Botões de Ação

O que cada Ação faz?

✏️ Editar

Editar agendamento completo

Abre o modal de edição — permite alterar dados do cliente, data, horário, status, OS, valores e mais.

✓ Compareceu

Marcar presença positiva

Registra que o cliente apareceu. Muda o status para "Compareceu" e pinta a linha de verde/amarelo.

✗ Não Comp.

Marcar ausência

Registra que o cliente não apareceu. Linha fica vermelha. Ação irreversível sem edição.

📋 OS

Registrar / Atualizar OS

Abre o modal de Ordem de Serviço para informar número, datas de entrada, entrega, valor e status.

🗑️ Excluir

Excluir agendamento

Remove o registro permanentemente. Disponível apenas para Admin e Atendimento Central.

Exportação

Exportar Relatório CSV

1

Aplique os filtros desejados

Escolha período, loja, status ou qualquer combinação de filtros antes de exportar.

2

Clique em "Relatório Operacional"

O botão fica no topo do painel de acompanhamento de OS.

3

Abra no Excel ou Google Planilhas

O arquivo .csv baixa automaticamente com todos os dados filtrados: clientes, datas, OS, valores, status.

💡

O CSV exporta exatamente os registros visíveis na tabela. Se quiser exportar tudo, limpe os filtros antes de clicar no botão. Para exportar o financeiro, vá até o Módulo de Faturamento e clique em "Exportar faturamento".

Sincronização Automática

Indicador de Atualização

Verde — Dados atualizados

O sistema sincronizou com sucesso. Os dados exibidos estão atualizados há menos de 15 segundos.

Amarelo — Atualizando...

O sistema está buscando os dados mais recentes. Aguarde alguns segundos até voltar ao verde.

O sistema atualiza automaticamente a cada 15 segundos sem precisar recarregar a página. Você sempre verá a informação mais recente enquanto estiver com o sistema aberto.

Dicas Importantes

Boas práticas do Sistema

🔐

Não compartilhe seu e-mail

Cada acesso é individual. Não empreste seu e-mail para outros usuários.

⏱️

Registre a OS logo

Abra a OS assim que o cliente comparecer para o Lead Time ficar preciso.

✅

Confirme presença no dia

Marque "Compareceu" ou "Não Compareceu" sempre no dia do agendamento.

📅

Verifique conflitos

O sistema bloqueia horários duplicados no mesmo optometrista e loja.

📊

Use os filtros

Filtre por período e loja para ver apenas o que é relevante para você.

❓

Problemas?

Fale com o administrador do sistema. Não tente adivinhar se algo parecer errado.
